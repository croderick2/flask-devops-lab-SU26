## Usage
This is a test of Git/GitHub/Docker for Computing Tools for the Workforce.

## Details
Virtual Environment:
  Ubuntu
  
Install Requirements:
  Python3
  Docker
  Git
  
Running the App:
###
    source .venv/bin/activate
    pip install -r requirements.txt
    python app.py
    
    Check progress with:
      curl http://localhost:8080/api/health
      curl http://localhost:8080/api/config
###

## Structures
/api/health:
  Health diagnosis of the API.
/api/config:
  Configurations of the API.
/api/report:
  Reports generated by the API.
